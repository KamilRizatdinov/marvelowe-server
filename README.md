# MarweLove-server

## Prerequisites

1. Create virtualenv:
   ```bash
   python -m virtualenv venv
   ```
2. Activate it:
   ```bash
   source venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## How to run

```bash
cd marvelowe-server
uvicorn main:app --reload
```
